# 기능 개발 고민

- 특정 지자체 페이지가 업데이트되었는지 확인할 수 있는지?
- 일정을 등록해서 대쉬보드에서 확인할 수 있도록 (홈, 뉴스, 법령, 일정)
- 대쉬보드에서는 오늘 뉴스, 오늘 시행 법령, 오늘 일정을 한 눈에 볼 수 있도록 하고 각 항목 클릭 시 해당 탭으로 이동하도록 한다.

# 개선사항

- QThread: Destroyed while thread '' is still running
- Calendar가 fetch와 build가 같이 발생해서 달력 로딩이 느림 (cache가 효율이 떨어짐)
- Calendar 첫 로딩 시 build 시점이 늦어서인지 이미 저장되어 있던 일정 box가 이상한데 렌더링되어 있음
